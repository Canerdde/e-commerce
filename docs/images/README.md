# Screenshots

Place your project screenshots in this folder.

## Recommended Screenshots:

1. **homepage.png** - Landing page with hero section
2. **products.png** - Products listing page with filters
3. **product-detail.png** - Individual product detail page
4. **cart.png** - Shopping cart drawer
5. **checkout.png** - Checkout page
6. **admin-dashboard.png** - Admin dashboard
7. **admin-products.png** - Admin products management
8. **admin-stock.png** - Admin stock management
9. **admin-pricing.png** - Admin pricing & discounts

## Image Guidelines:

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or similar aspect ratio
- **Quality**: High resolution for better display on GitHub
- **Naming**: Use descriptive names (kebab-case)

## How to Add Screenshots:

1. Take screenshots of your application
2. Save them in this folder (`docs/images/`)
3. Update the README.md file to reference the images:
   ```markdown
   ![Homepage](./docs/images/homepage.png)
   ```

## Alternative: Using GitHub Issues/PRs

You can also host images on:
- GitHub Issues (drag & drop)
- Imgur
- Cloudinary
- Or any image hosting service

Then use the full URL in README:
```markdown
![Homepage](https://your-image-url.com/homepage.png)
```
